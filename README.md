## Installation

2. Place the downloaded `.jar` file into the `plugins` folder of your Minecraft server.
3. Restart the server to load the plugin.

## Submitting to Modrinth and SpigotMC
1. For [Modrinth](https://modrinth.com):
   - Create an account on Modrinth.
   - Navigate to the "Upload" section.
   - Fill in the necessary information about your plugin and upload the `.jar` file.

2. For [SpigotMC](https://www.spigotmc.org):
   - Create an account on SpigotMC.
   - Go to the "Resources" section and click "Add Resource".
   - Provide the required details and upload your plugin file.
